1️⃣ Overview — What Is This Tool?

The MicrOrbit · Project Management System is a purpose-built operational dashboard for the MicrOrbitperations team within T&I. It enables real-time tracking of project workplans, timelines, ownership, and progress in a single browser-based application accessible directly from SharePoint.

Open the link and it works — no installation, no IT tickets, no additional software. Data is stored in a shared cloud workspace so your whole team sees the same live information the moment anything changes.

  • Add, edit, and delete projects directly — no spreadsheets required
  • 7 professional themes switchable instantly from the sidebar
  • Smart recommendations auto-generated from your live data
  • Full team cloud sync — every change visible to all within seconds
  • Professional reports generated and distributed in one click
💡 This dashboard operates like any enterprise web application — open a link, use it, close it. On SharePoint it gets a permanent URL your entire team can bookmark.

2️⃣ The Work & Effort Behind This Dashboard

This dashboard was not assembled from a template or purchased off-the-shelf. It was designed, engineered, tested, and refined from scratch over approximately 18–25 hours of focused, skilled technical development across multiple dedicated sessions.

Effort breakdown:

  • Requirements & Planning (~2 hrs) — Analysing your team's workflow, mapping the workplan structure, defining every required feature
  • UI/UX Design System (~3 hrs) — Designing 7 distinct professional themes, full navigation architecture, component library, and typography from scratch
  • Frontend Engineering (~8 hrs) — Hand-writing all HTML, CSS, and JavaScript — 11-page navigation, responsive layout, theme engine, animations, and all interactions
  • Data Engine & Visualizations (~3 hrs) — Excel parser, data normalization pipeline, 6 Chart.js visualizations, and Gantt renderer
  • Cloud Backend Integration (~2 hrs) — Shared cloud sync layer, real-time push/pull, auto-sync, and secure credential handling with no UI exposure
  • Recommendations Engine (~2 hrs) — Smart insights algorithm surfacing portfolio health scores, risk ratings, and prioritized actions
  • Reporting System (~2 hrs) — Report generation pipeline with email, HTML download, and clipboard copy
  • Testing & Refinement (~3 hrs) — End-to-end testing, design polish across multiple sessions, SharePoint deployment preparation

For context — a tool of this scope through formal IT procurement at the commonly takes 3–6 months before a single line of code is written, covering business requirements, IT security review, architecture board approval, vendor RFP, contract negotiation, multi-sprint development, user acceptance testing, and training rollout.

Comparable platforms teams subscribe to include Monday.com ($120–400/month), Smartsheet ($160–500/month), and Microsoft Project Online ($500+/month) — none customized to your specific workflow the way this tool is.

💡 Every feature in this planner exists because your team specifically needed it — purpose-built for Operations, not adapted from a generic template.

3️⃣ Adding & Managing Projects

Go to ➕ Add / Edit Projects in the sidebar. No spreadsheets needed.

  • Add — complete the form and click Add Project. Appears instantly everywhere
  • Edit — click ✏️ Edit on any row, update the pre-filled form, click Save Changes
  • Delete — click 🗑 Delete. Confirmation required before removal
  • Cloud sync — every change auto-pushes to the shared workspace within seconds
💡 Status options: On Track · In Progress · Overdue · On Hold · Not Started · Complete. Keep Progress % and Status synchronized for accurate KPIs and Recommendations.

4️⃣ Dashboard — KPI Cards

Seven live metric cards update automatically: Total Projects · On Track · Overdue · Complete · On Hold · Avg Progress · Team Leads. Below the cards, a status doughnut and lead bar chart provide an instant visual summary. The full project table appears beneath.

💡 Cards with red or amber accents indicate areas needing attention.

5️⃣ Charts & Analytics

Five interactive charts — hover any element for exact values: Status Breakdown · Projects by Lead · Overall Completion · Support Team Load · Progress by Project (colour-coded by performance level). Use Progress by Project before stakeholder presentations for instant clarity on which initiatives are lagging.

6️⃣ Recommendations Engine

The 💡 Recommendations page automatically analyses live data and surfaces: clickable Alert Banners for critical issues, specific Action Items with context, a 0–100 Portfolio Health Score with circular gauge (Healthy/At Risk/Critical), and a Deadline Risk Table with 🔴🟠🟡🟢 ratings.

💡 Review this page every Monday and before any leadership meeting. It tells you exactly where to focus without manually reading every project entry.

7️⃣ Timeline / Gantt View

All projects shown as proportional Gantt bars scaled to actual dates. A red line marks today. 🔵 Blue = On Track · 🔴 Red = Overdue · 🟠 Amber = On Hold · 🟢 Green = Complete. Scroll horizontally for wide date ranges.

8️⃣ Task List & Filters

Searchable, sortable, filterable full project table. Click any column header to sort. Combine filters — e.g. Lead "ESI" + Status "Overdue" shows only ESI's overdue projects. Click Clear to reset all filters.

9️⃣ Milestones & Deadlines

All active projects sorted soonest-first by end date. Days Left column counts calendar days from today — negative = already overdue. Colour coded 🟢🟡🔴. Review before every team meeting to catch deadlines early.

🔟 Sending Reports

Three export options: Open in Email (launches Outlook with pre-written report) · Download HTML (open in browser → Ctrl+P → Save as PDF for formal distribution) · Copy Text (instant clipboard for Teams or Outlook). Set recipients once — they stay saved.

1️⃣1️⃣ Themes & Admin Settings

Seven themes switchable instantly from the sidebar dots or Admin → Appearance: Light Corporate · Dark Mode · Modern Blue · Emerald Green · Minimal Grey · Executive Purple · High Contrast. Admin also provides CSV/JSON data export, column toggles for the Task List, and a system information panel.

💡 Use Dark Mode or Executive Purple for screen presentations. Use High Contrast for maximum accessibility and readability.

1️⃣2️⃣ FAQ & Troubleshooting

  • How do I add a project? Add / Edit Projects → complete the form → Add Project.
  • Will my team see changes? Yes — auto-syncs to shared workspace within seconds.
  • No data on open? Go to Add / Edit Projects → Pull Latest to manually refresh from cloud.
  • How do I switch themes? Click any dot at the bottom of the sidebar, or Admin → Appearance.
  • Charts are blank? Navigate away and back, or ensure at least 2 projects exist.
  • How to backup data? Admin → Data → Export to JSON. Restore anytime from the same tab.
  • How to share with team? Upload .aspx to SharePoint SiteAssets → copy link → distribute.
  • How to print a PDF report? Send Report → Download HTML → open in browser → Ctrl+P → Save as PDF.
